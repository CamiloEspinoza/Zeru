# Closing Checklist — mensual/anual (referencia)

## Objetivo
Checklist práctico para cierre contable en sistema.

## Cierre mensual (mínimo)
- [ ] Periodo creado y con estado `open`
- [ ] Todos los documentos del periodo procesados (ventas/compras/pagos)
- [ ] Revisión de asientos en borrador/errores
- [ ] Conciliación bancaria (completa o con pendientes documentados)
- [ ] CxC vs cuenta control
- [ ] CxP vs cuenta control
- [ ] Provisiones/reclasificaciones del mes
- [ ] Depreciación del mes (si aplica)
- [ ] Validación balance de comprobación
- [ ] Resumen IVA compras/ventas revisado
- [ ] Snapshot de saldos por cuenta (opcional recomendado)
- [ ] Cambio de estado a `soft-closed` / `closed`
- [ ] Bloqueo de posteo retroactivo no autorizado

## Reapertura de periodo
- [ ] Usuario autorizado
- [ ] Motivo registrado
- [ ] Audit log creado
- [ ] Reproceso controlado de reportes afectados

## Cierre anual (adicional)
- [ ] Cierre de resultado del ejercicio (según política)
- [ ] Preparación de bases para tributario/anual
- [ ] Arrastre de saldos a nuevo año
- [ ] Congelamiento de reportes emitidos
