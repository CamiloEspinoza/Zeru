Use these templates to produce consistent, audit-ready outputs.
Adapt depth to materiality and user intent.

## 3) Transaction Review Checklist

Use when the user wants a practical checklist for controller, finance manager, or audit prep.

### Template

# IFRS Transaction Review Checklist: [Transaction Name]

## A. Facts to Confirm
- [ ] Signed contracts / amendments obtained
- [ ] Reporting period and cut-off date confirmed
- [ ] Parties and related-party status confirmed
- [ ] Payment terms and variable consideration confirmed
- [ ] Management intent documented (if relevant under IFRS)
- [ ] Materiality threshold / amount quantified

## B. Scope and Literature Mapping
- [ ] Primary standard identified
- [ ] Scope exclusions tested
- [ ] Supporting standards identified
- [ ] Interpretation guidance considered (if relevant)

## C. Accounting Conclusions to Finalize
- [ ] Recognition timing
- [ ] Initial measurement
- [ ] Subsequent measurement / impairment / remeasurement
- [ ] Presentation / classification
- [ ] Disclosure set

## D. Calculations / Models Needed
- [ ] Measurement workbook
- [ ] Discounting / valuation support
- [ ] Impairment / ECL analysis
- [ ] Allocation schedules
- [ ] Sensitivity analysis (if material)

## E. Documentation / Audit Evidence
- [ ] Contracts and legal docs
- [ ] Valuation reports / market evidence
- [ ] Management approvals
- [ ] Assumptions and judgment memos
- [ ] Control evidence (review/sign-off)

## F. Reporting Outputs
- [ ] Journal entries posted / reviewed
- [ ] Disclosure draft prepared
- [ ] Policy alignment confirmed
- [ ] Comparative impact assessed
- [ ] External auditor discussion prepared (if needed)
